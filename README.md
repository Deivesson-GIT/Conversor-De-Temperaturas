#  Conversor de Temperaturas
 Objetivo do Projeto
O objetivo deste projeto é desenvolver uma aplicação web interativa e responsiva que permita ao usuário converter valores entre diferentes escalas de temperatura (Celsius, Fahrenheit e Kelvin).  
O projeto foi criado para consolidar os conhecimentos em **HTML5, CSS3 e JavaScript**, aplicando boas práticas de estruturação, estilo, interatividade e versionamento de código.

 Funcionalidades
- Conversão entre Celsius, Fahrenheit e Kelvin.
- Validação de entrada para evitar erros de cálculo.
- Interface simples, responsiva e intuitiva.
- Feedback visual com resultado formatado.
- Navegação básica com cabeçalho e rodapé.

Tecnologias Utilizadas
- **HTML5** → Estrutura semântica da página.  
- **CSS3** → Estilização, responsividade e efeitos visuais.  
- **JavaScript (Vanilla)** → Lógica de conversão e manipulação do DOM.  
- **Git/GitHub** → Versionamento e publicação do projeto.
